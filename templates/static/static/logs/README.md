# Internet Speed Tester

A techy, eye-catching web app to measure internet download speed in MB/s.

## Features
- Real-time speed test with progress bar and animated gauge.
- Dark theme with effects.
- Client-side JS for testing, Flask backend for serving.

## Setup
1. Clone the repo.
2. Install dependencies: `pip install -r requirements.txt`
3. Run: `python app.py`
4. Open `http://localhost:5000`

## Deployment
- Use Heroku, Vercel, or GitHub Pages (for static parts; backend needs a server).
- For PHP logging, upload to a PHP host.

## Tech Stack
- Python (Flask)
- HTML/CSS/JS
- PHP (optional)
